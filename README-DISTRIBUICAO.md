# MEEM Game - Versão de Distribuição

## Como Executar

### Opção 1: Usar o script start.bat (Recomendado)
1. Dê duplo clique no arquivo `start.bat`
2. O jogo abrirá automaticamente em: http://localhost:3000
3. Pressione Ctrl+C para parar o servidor

### Opção 2: Usar npx serve manualmente
1. Abra o terminal nesta pasta
2. Execute: `npx serve dist -p 3000`
3. Acesse: http://localhost:3000

### Opção 3: Abrir diretamente no navegador
1. Abra o arquivo `dist/index.html` no seu navegador
2. Nota: Algumas funcionalidades podem não funcionar corretamente sem um servidor

## Requisitos
- Node.js instalado (para usar o script start.bat)
- Navegador web moderno (Chrome, Firefox, Edge, Safari)

## Funcionalidades
- 11 blocos de testes cognitivos
- Exportação de resultados para Excel
- Armazenamento local de resultados
- Interface responsiva com tema violeta
